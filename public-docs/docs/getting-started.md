# Getting Started

This guide will help you set up RGFX for the first time.

## Requirements

- **RGFX Hub** - macOS application (download from releases)
- **ESP32 microcontroller** - ESP32-C3 or ESP32-S3 recommended
- **LED hardware** - WS2812B LED strips or matrices
- **MAME** - Version 0.250 or later

## Quick Start

1. **Install RGFX Hub** - Download and install the Hub application
2. **Flash a Driver** - Use the Hub to flash firmware to your ESP32
3. **Configure LEDs** - Set up your LED strip or matrix configuration
4. **Connect to MAME** - Install interceptors for your favorite games
5. **Play!** - Start a game and watch the lights react
