# Acknowledgements

![Thank you](assets/images/thank-you.gif)

A personal thank you to everyone in the retro-gaming community who has inspired me to create this project. The passion for preserving and celebrating arcade and home console history keeps video games alive in ways their creators never imagined.

Special thanks to the organizers of [Portland Retro Gaming Expo](https://www.retrogamingexpo.com/) and [Midwest Gaming Classic](https://www.midwestgamingclassic.com/) for keeping the retro game spirit thriving.

---

RGFX is built on the shoulders of many outstanding open-source projects. Thanks to all the developers and communities behind them.

## Emulation

| Project | Description |
|---------|-------------|
