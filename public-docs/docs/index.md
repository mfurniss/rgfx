# RGFX Documentation

Welcome to the RGFX documentation. RGFX (Retro Game Effects) is a framework that creates external visual effects for retro video games.

## What is RGFX?

RGFX connects your retro gaming setup to LED lighting effects. When you play classic arcade games in MAME, game events like scoring points, losing lives, or defeating enemies trigger real-world LED effects on strips and matrices connected to ESP32 microcontrollers.

## Components

The RGFX system consists of three main components:

- **RGFX Hub** - The desktop application that coordinates everything
- **ESP32 Drivers** - Microcontroller firmware that controls LED hardware
- **MAME Interceptors** - Lua scripts that monitor game state and generate events
