# System Requirements

## Hardware

* Apple Mac or Windows PC
* ESP32 development board(s)
  * ESP32-WROOM-32, ESP32-S2, ESP32-S3, or ESP32-C3
  * _Compatible ESP32 LED drivers may be used if they support flashing custom firmware_

  
* WS2812B RGB LED strips / matrices
* Local WiFi network

## Software

* RGFX app package
* MAME app version 0.250 or later
  * [macOS version](https://sdlmame.lngn.net)
  * [Windows version](https://www.mamedev.org/release.html)
* Optional - Your favorite emulation frontend / launcher
  * _The frontend must support launching the external MAME app_
